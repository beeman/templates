# Templates

This repository serves as a collection of templates for new projects.

## Available Templates

### `bun-library`

A modern TypeScript library or package template built with [Bun](https://bun.sh/). It comes pre-configured with essential tools for development, testing, linting, and publishing.

To use this template, run the following command:

```bash
bun x create-solana-dapp@latest -t gh:beeman/templates/bun-library
```

This will create a new project in a directory of your choice, clone this template into it, and install the dependencies.
