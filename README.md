# Templates

Project templates for create-seed.

## Available Templates

### `bun-convex-tanstack`

Bun monorepo template with Turbo, Convex and Tanstack Router web app

```bash
bun x create-seed@latest my-app -t gh:beeman/templates/bun-convex-tanstack
```

### `bun-library`

TypeScript library with Bun, tsup, Biome, and Changesets

```bash
bun x create-seed@latest my-app -t gh:beeman/templates/bun-library
```

### `bun-library-solana-kit`

TypeScript library for Solana development with @solana/kit, Surfpool e2e tests, Bun, tsup, Biome, and Changesets

```bash
bun x create-seed@latest my-app -t gh:beeman/templates/bun-library-solana-kit
```

### `bun-monorepo`

Bun monorepo template with Turbo, Hono API, Biome, and Docker support

```bash
bun x create-seed@latest my-app -t gh:beeman/templates/bun-monorepo
```
